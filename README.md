# Image Deblurring: A Technical Overview

A technical overview of image deblurring methods, covering the Bayesian 
formulation of non-blind deblurring, hyper-Laplacian priors, learning-based 
approaches (NAFNet, Restormer, FFTformer, HiDiff), and evaluation metrics 
(PSNR, SSIM, LPIPS).

## Topics Covered
- Problem formulation: blind vs. non-blind deblurring
- Traditional methods: Bayesian MAP estimation, hyper-Laplacian priors
- Limitations of traditional methods (ill-posed, spatially-variant blur)
- Learning-based methods: CNN, Transformer, Diffusion-based approaches
- Evaluation metrics: PSNR, SSIM, LPIPS
